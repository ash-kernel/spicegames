## ✨ Features

### 🎮 Game Library
- Add any game (`.exe`)
- Clean, organized library
- One-click launch

### 🧠 Smart Metadata
- Covers, screenshots, descriptions  
- Genres & tags  
- Steam, SteamSpy, Metacritic data  
- **HowLongToBeat integration**

### 📊 Stats & Insights
- Playtime tracking + session history  
- Charts (daily, genre)  
- Ratings from Steam, Metacritic, OpenCritic  

### 📚 Collections
- Favorites, Playing, Backlog, Completed, Wishlist  
- Status tracking + notes  
- Sorting & filtering  

### 🔎 Discover
- Browse 50K+ games  
- Filter by genre, trending, top-rated  
- Search + previews  

### 📰 News
- PC Gamer, IGN, Eurogamer & more  

### 💰 Deals
- Steam, Epic, GOG, Humble  
- Multi-currency support  
- Filters by price & discount  

### 🎮 Extras 
- Controller support & tester  
- Screenshot scanner  
- Widget mode  

### ⚙️ System
- Auto-update checker  
- System tray support  
- Optional startup launch  
- Fully offline — no accounts
